# 🛡️ Intrusion Detection System (IDS)

A modular **Host-Based Intrusion Detection System (HIDS)** developed in Python for monitoring system activities and detecting suspicious behavior. This project is designed for educational purposes to demonstrate the fundamentals of intrusion detection, file integrity monitoring, process monitoring, network observation, and security reporting.

---

## 📌 Features

* Real-time file integrity monitoring
* Process monitoring
* Active network connection monitoring
* Packet monitoring (planned)
* Suspicious activity detection
* SHA-256 file hashing
* JSON and HTML report generation
* Event logging
* Modular architecture
* Simple command-line interface
* Cross-platform Python implementation

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YourUsername/Intrusion-Detection-System.git
```

Move into the project directory:

```bash
cd Intrusion-Detection-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
